This directory holds pre-generated files for cross-compilation targets.

`janet_core_image.c` is generated by the Janet bootstrap build (native targets) and
committed here so the WASM build can compile Janet without running a native executable.

To regenerate after updating the Janet submodule:

```bash
cd packages/dear-imgui/cpp/tests
cmake -S . -B build
cmake --build ./build --target janet_core
cp build/janet_core_image.c ../app/generated/
```
